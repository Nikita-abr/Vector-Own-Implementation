# Vector — упрощённый аналог `std::vector`

Учебный проект на C++, представляющий собой реализацию упрощённого аналога `std::vector<int>`


Цель проекта — демнострация уверенного понимания:
- ручное управлением памятью, работа с ресурсами
- семантики копирования и перемещения
- устройство STL-совместимых итераторов
- внутреннее устройство std::vector

## Возможности

* Хранение элементов типа `int`
* Динамическое управление памятью (`Size`, `Capacity`)
* Автоматическое расширение(реалокация) при `PushBack`
* Поддержка:

  * конструктора по умолчанию
  * конструктора с размером
  * `std::initializer_list`
  * копирующего и перемещающего конструкторов
  * копирующего и перемещающего присваивания
* Методы:

  * `PushBack`, `PopBack`, `Clear`, `Reserve`, `Swap`
* Операторы индексирования (`const` и неконстантный)
* STL-совместимые **random access итераторы**
* Поддержка `range-based for`

## Пример использования

```cpp
Vector v = {1, 2, 3};
v.PushBack(4);

for (int& x : v) {
    x *= 2;
}
```
